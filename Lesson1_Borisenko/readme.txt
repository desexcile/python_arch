WSL Убунту уже была установлена, часто использую ее в работе и личных целях. ConEmu в качестве терминала.
Python 3.6 тоже был уже на убунте, поэтому доставил uwsgi через:
	sudo pip3 install uwsgi
затем поставил само приложение:
	sudo apt-get uwsgi
запустилось все без проблем командой: 
	sudo uwsgi --http :8000 --wsgi-file fwsgi.py
в проекте лежит файл лога запуска uwsgi.

Однако, решил использовать wsgiref, так как проще запускать и ваш проект на нём, работать с ним проще чем с WSL машиной в конкретной ситуации.
С ним проблем не возникло тоже, все запустилось и работает.
